# VidyaGuide AI Agent – Career Planning & Resume Mentor

## Project Overview
VidyaGuide AI Agent is an AI-powered platform designed to assist students in improving their resumes and planning their careers effectively. The system analyzes a user's resume, identifies key skills, detects skill gaps, and provides personalized career recommendations. Using Large Language Models (LLMs), the platform offers intelligent insights, learning roadmaps, and career guidance to help students enhance their professional profiles.

This project helps bridge the gap between academic knowledge and industry requirements by providing automated resume evaluation and AI-based career mentoring.

---

## Key Features
- Resume Analysis – Upload a resume and extract important information such as skills and qualifications.
- Skill Gap Detection – Identify missing skills required for different career paths.
- Career Recommendations – Suggest suitable career options based on the user's profile.
- Learning Roadmap – Provide guidance on courses and skills to improve career opportunities.
- AI Career Mentor – Interactive AI assistant to answer career-related queries.
